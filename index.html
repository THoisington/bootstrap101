CODEcamp - Bootstrap 101

Course Overview

Why should I take Bootstrap 101?

Bootstrap is a mobile-first, front-end toolkit for faster and easier web development. It contains HTML and CSS-based design templates for typography, forms, buttons, tables, navigation and other interface components, as well as several optional JavaScript extensions. Fully extensible and documented, Bootstrap makes it simple to develop user-friendly, professional websites and web applications.

What will I learn?

The class will consist of an overview of Bootstrap, followed by a walk through of a completed website. Using this site as a guide, you will then create your own site through a series of hands-on exercises to ultimately learn by doing.

What can I expect?

Each 2.5-hour session is a mixture of Lecture and hands-on Exercises to best teach you the ins and outs of using Bootstrap. Hands-on practice is essential to understand and fully grasp the concepts presented. The class is limited to 14 students.

Class 1 - The Basics: What is Bootstrap and how to get started

In this class you will learn everything you need to know about what Bootstrap is, what it is used for, and the benefits and drawbacks of using Bootstrap over other CSS frameworks. After that, you will receive assistance in setting up all the tools necessary to get coding with Bootstrap. Your instructor recommends using the text editor Sublime and linking in Bootstrap by using their CDN links found on getbootstrap.com.

The remainder of the class will be spent on the CSS page of Bootstrap's website, studying CSS settings of commonly-used HTML elements while learning the general semantics and quirks of Bootstrap in the process. Your instructor will code examples for each HTML element that students will reciprocate on their own computers.
